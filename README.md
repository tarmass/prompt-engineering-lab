# Prompt Engineering Lab

Daily exercises to build systematic prompt engineering skills. 15-20 minutes a day, 5 days a week.

## Structure

```
exercises.md       — The daily exercise framework (Mon-Fri rotation)
daily-log/         — Daily prompt + output + what I learned
week-01/           — Weekly portfolio pieces (Friday builds)
```

## Weekly Rotation

| Day | Focus |
|-----|-------|
| Monday | Constraint Engineering |
| Tuesday | System Prompt Architecture |
| Wednesday | Evaluation & Debugging |
| Thursday | Chain-of-Thought & Decomposition |
| Friday | Portfolio Piece |

## What This Builds

- Making models reliably follow complex instructions
- Debugging when outputs go wrong
- Understanding failure modes (hallucination, sycophancy, instruction drift)
- Designing evaluation criteria
- Building multi-step systems, not just one-shot prompts

## How to Use

1. Open `exercises.md` for today's exercise
2. Do the exercise (15-20 min)
3. Save your work to `daily-log/YYYY-MM-DD.md`
4. Friday: build a portfolio piece in `week-XX/`
